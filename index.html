NODE JS

A runtime environment that lets you run JavaScript on the server. Used to build fast, scalable backend applications and APIs. Built on Google Chrome’s V8 engine for high performance. MONGO DB

A NoSQL, document-based database that stores data in JSON-like format (BSON). Great for flexible, schema-less data models that scale easily. Used with Node.js in the popular MERN stack (MongoDB, Express, React, Node). Supports powerful querying, indexing, and aggregation features. Ideal for real-time apps and dynamic data structures.
